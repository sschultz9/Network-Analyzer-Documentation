Selfscan Module¶

Module containing local scanning functionality similar to netstat.

The selfscan module will find listening TCP/UDP sockets on the local machine. This module scans through /proc and finds data pertaining to the listening sockets. This includes the inode number, process ID, command that created the listening socket, protocol, local and remote IP addresses with associated ports, and the state of the connection.

selfscan._enumerate_pids()¶

Enumerates the list of pids in /proc for command line info and maps them to the correct inode.

Returns:

Inodes dictionary, populated with pid and command info.

Return type:

dict

selfscan._get_proc_net(path: str, ip_state: str) → dict¶

Gets the local address, remote address, state, and inode data of each listening connection.

Parameters:

  • path (str) – Absolute path to the file to be checked.

  • ip (str) – String representing whether the protocol is IPv4, or IPv6.

Returns:

Dictionary of connections, mapping each inode to {proto, ip, port, state} for all 0A (listening) connections.

Return type:

dict

selfscan._hex_conversion(hex_string: str, ip: str) → str¶

Takes a little endian hex string and converts it to an IP address.

Parameters:

  • hex_string (str) – String with the format <IP>:<port> in hex.

  • ip (str) – String indicating the IP version to use for conversion.

Returns:

The <IP>:<port> string, converted to base 10 format.

Return type:

str

selfscan._list_inodes(inodes_dict: dict, conns: list)¶

Prints out the list of inodes and their associated data.

Parameters:

  • inodes_dict (dict) – Dictionary mapping inodes and their data.

  • conns (list) – List containing the dicts for TCP/UDP v4 and v6.

selfscan.scan_local_device()¶

Performs a netstat-like listening scan on the local device.

Verifies that /proc exists. Then, parses /proc/net for tcp and udp information. Enumerates PIDs and gathers socket info per inode.

At the end, the utility will print out a list of listening connections and relevant data associated with each connection.
