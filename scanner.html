Scanner Module¶

Module containing network scanning functionality.

The scan function can perform a variety of scans on a target network, which is specified via CIDR notation. If the scan type is a SYN scan, then the scan will enumerate a list of ports on each target host as specified by the user. Each port is sent a SYN packet and if a SYN/ACK is received as a response the port is listed as open, if a RST is received as a response the port is marked as closed, and if an ICMP Destination Unreachable packet is received or there is no response, then the port is listed as being filtered. The ICMP scan is much simpler as it just sends an ICMP Echo Request packet to every target host and lists the hosts that responded as being alive.

Typical usage example:

scan(“192.168.1.0/24”, ScanType.ICMP, (22, 23, 80, 443), 0.5)
