Scanner Module¶

Module for the Scanner class.

The scanner class can perform a variety of scans on a target network, which is specified via CIDR notation. If the scan type is a SYN scan, then the user can specify a port list to scan on each target host, if no port list is specified then a default port list will be utilized in the scan.

Typical usage example:

scanner = Scanner(0.1) scanner.scan(“192.168.1.0/24”, ScanType.ICMP)
