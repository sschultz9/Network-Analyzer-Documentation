Main Module¶

Main module for parsing user input and executing a network scan.

This module will read command line input from the user, and perform a network scan based on the user’s specified configuration.

main.main()¶

Collects network scanning variables from user input with argparse.

main.verify_cidr(target: str) → None¶

Verifies the passed in target IP address is a valid CIDR IP address.

Parameters:

target (str) – IP address in CIDR notation.

Raises:

ValueError – If the target IP address is invalid or not in CIDR notation.

main.verify_ports(ports: str) → tuple[int, ...]¶

Verifies that the user entered valid port values.

Parameters:

ports (str) – A comma-separated list of port values.

Returns:

A tuple of integers representing the ports to be scanned.

Return type:

tuple[int, …]

Raises:

ValueError – If any of the port values are not integers or are not in the range of 0-65535.

main.verify_scan_type(scan_type: str) → ScanType¶

Verifies that the user entered a valid network scan type.

Parameters:

scan_type (str) – The network scan type to be checked.

Returns:

Enum representing the user’s chosen network scan type.

Return type:

ScanType

Raises:

ValueError – If the user specified network scan type is unrecognized.
