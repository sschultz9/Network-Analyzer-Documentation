Main Module¶

Main module for parsing user input and performing network analysis.

This module will read command line input from the user, and subsequently perform the user specified network analysis.

main.do_not_scan(file_path: str) → list[str]¶

Parses the specified file for IP addresses and CIDR ranges to blacklist.

Parameters:

file_path (str) – Path (relative or absolute) to the file which contains the IP addresses and CIDR ranges which should not be scanned.

Raises:

ArgumentTypeError – If the file at the provided path cannot be opened or read.

Returns:

A list of IP addresses which will not be scanned during the network scan.

Return type:

list[str]

main.main()¶

Collects network analysis variables from user input using argparse.

main.verify_ip(ip: str) → str¶

Verifies the passed in target IP address is a valid CIDR IP address.

Parameters:

ip (str) – The IP address to be checked.

Raises:

ArgumentTypeError – If the passed in IP address is invalid or not in CIDR notation.

main.verify_max_hops(max_hops: str) → int¶

Verifies that the user provided max hops value is between 0 and 255.

Parameters:

max_hops (str) – The max hops value to be checked.

Returns:

Integer representation of the user provided max hops value.

Return type:

int

Raises:

ArgumentTypeError – If the max hops value cannot be converted to an integer or is not between 0 and 255.

main.verify_ports(ports: str) → tuple[int, ...]¶

Verifies that the user provided a valid string of port values.

Parameters:

ports (str) – A comma-separated list of port values, either single-valued or given as a range from a lower bound to an upper bound, inclusive.

Returns:

A tuple of integers representing the ports to be scanned.

Return type:

tuple[int, …]

Raises:

ArgumentTypeError – If any of the argument port values are not integers or are not in the range of 0-65535 (The range of valid port numbers).

main.verify_timeout(timeout: str) → float¶

Verifies that the user provided timeout value is non-negative.

Parameters:

timeout (str) – The timeout value to be checked.

Returns:

Floating point representation of the user provided timeout value.

Return type:

float

Raises:

ArgumentTypeError – If the timeout value is negative or cannot be converted to a float.
