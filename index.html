Network Analyzer documentation¶

Command-line Flags¶

-t, --target <CIDR>¶

Target IP address range in CIDR notation for the network scan.

-s, --scan_type {SYN,ICMP}¶

The type of network scan to be performed.

-p, --ports <ports>¶

Comma-separated list of port numbers to enumerate. This is only allowed if the selected scan type is a SYN scan. If no ports are specified, a default set of ports will be enumerated.

Example usages¶

sudo python3 main.py -t 192.168.0.0/24 --scan_type icmp
sudo python3 main.py -t 127.0.0.1/32 -s SYN -p 22
sudo python3 main.py --target 192.168.0.0/30 -s ICMP
sudo python3 main.py --target 192.168.1.0/24 --scan_type SYN --ports 20,21,22
sudo python3 main.py --target 192.168.1.0/24 --scan_type SYN
