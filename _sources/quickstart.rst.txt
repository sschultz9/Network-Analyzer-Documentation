Quickstart Guide
================

This quickstart guide assumes you are on a modern x64 Linux environment.

1. Clone Network-Scanner
------------------------

The first step is to clone the repository.

.. code:: bash

   git clone https://github.com/sschultz9/Network-Analyzer.git

2. Setup virtual environemnt
============================

This program was built in Python 3.13.5 and expects the user to install
Python 3.13.5 to ensure compatibility. To build the environment, you
must also have the python virtual environment module installed.

Once you have the venv module installed, run the commands listed below
in your terminal:

1. Change directories to the root of the repository.

.. code:: bash

   cd Network-Analyzer

2. Create a new python virtual environment to run the program in.

.. code:: bash

   python3 -m venv env

3. Activate the newly created virtual environment

.. code:: bash

   source env/bin/activate

4. Install the required packages

.. code:: bash

   pip install -r requirements.txt

Executing program
~~~~~~~~~~~~~~~~~

1. Change directories to the src folder that contains main.py

.. code:: bash

   cd Network-Analyzer/src

2. This is one example of how you could execute the program. Refer to
   our
   `documentation <https://sschultz9.github.io/Network-Analyzer-Documentation/>`__
   website for further information regarding the various configurations
   you can use to run our program.

.. code:: bash

   python3 main.py --target 192.168.1.0/24 --scan_type SYN --ports 22,80,443
