Quickstart Guide¶

1. Clone the repository¶

The first step is to clone the repository

git clone https://github.com/sschultz9/Network-Analyzer.git

2. Setup a virtual environemnt¶

This program was built in Python 3.13.5 and expects the user to install Python 3.13.5 to ensure compatibility. To build the environment, you must also have the python virtual environment module installed. Once you have the venv module installed, run the commands listed below in your terminal:

  1. Change directories to the root of the repository

cd Network-Analyzer

  1. Create a new python virtual environment to run the program in

python3 -m venv venv

  1. Activate the newly created virtual environment

source venv/bin/activate

  1. Install the required packages

pip install -r requirements.txt

3. Execute the program¶

  1. Change directories to the src folder that contains main.py

cd src

  1. This is one example of how you could execute the program. Refer to this website’s home page for further information pertaining to various command-line flags that can be utilized

python3 main.py --target 192.168.1.0/24 --scan_type SYN --ports 22,80,443
